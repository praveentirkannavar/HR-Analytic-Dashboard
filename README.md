👥 HR Analytics Dashboard (Power BI)
📊 Overview

This project showcases an interactive HR Analytics Dashboard built using Microsoft Power BI.
It helps organizations analyze employee data, track attrition, and gain insights into workforce trends to support better decision-making.

🎯 Key Objectives
Analyze employee attrition
Understand workforce distribution
Identify factors affecting employee retention
Provide actionable HR insights


✨ Features
📉 Attrition Analysis (overall and department-wise)
👨‍💼 Employee Demographics (age, gender, education)
🏢 Department & Job Role Insights
💰 Salary & Compensation Analysis
⏳ Years at Company vs Attrition
📊 Job Satisfaction & Performance Metrics
🖥️ Dashboard Components
KPI Cards:
Total Employees
Attrition Count
Attrition Rate (%)
Average Salary
Charts:
Bar charts (Department, Job Role)
Pie/Donut charts (Attrition distribution)
Line charts (Trends over time)
Histogram (Age distribution)

🔐 Authentication Note

When opening the report in Microsoft Power BI, you may see a Windows Security passkey sign-in prompt.

Only work or school accounts are supported
Personal emails (Gmail, Yahoo, Outlook personal) may not work
You can sign in using:
QR code (passkey via phone)
Microsoft Authenticator
Organization credentials
